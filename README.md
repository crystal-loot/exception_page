# exception_page

A library for displaying exceptional exception web pages for easier debugging.

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  exception_page:
    github: crystal-loot/exception_page
```

## Usage

```crystal
require "exception_page"
```
